Formula Editor: This allows users to define a formula that maps the response variable to the predictor variable(s) for analysis.

Data Grouping and Splitting: Users can dynamically separate data into groups or apply filters, which is reflected in the UI: Split by Group: Enables users to subset data based on one or more column(s).

Confidence Level: A slider to set the confidence level for the interval, ranging from 0 to 1 (default is 0.95).

Alternative Hypothesis: A dropdown menu for selecting the type of alternative hypothesis (two-sided, less, or greater).

Two groups:

  • Paired or Unpaired: Select whether the t-test should be paired or unpaired.
  • Variance Equality: Decide if the variances of the two groups are equal or not.

Multiple Group Tests: For cases with more than two groups, users can perform:

  • ANOVA: Conduct an analysis of variance test.
  • Kruskal-Wallis Test: Non-parametric test for comparing more than two groups.

Post Hoc Tests: After conducting ANOVA, users can select and run post hoc tests, including:

  • Tukey HSD
  • Kruskal Wallis post hoc test
  • Least Significant Difference test (LSD)
  • Scheffe post hoc test
  • REGW post hoc test
Users can also adjust the p-value threshold and choose between balanced or unbalanced designs.

Adjusted p-Value Method: When applicable, users can select the method for adjusting p-values for multiple comparisons, including:

  • Holm
  • Hommel
  • Hochberg
  • Bonferroni
  • BH (Benjamini-Hochberg)
  • BY (Benjamini-Yekutieli)
  • FDR (False Discovery Rate)

Saving and Exporting Options

Once satisfied with the results, use these controls to save and export:

  • Add to Result File: Adds the current result to the result file. This is a temporary file holding all results.
  • Final selection: tick the boxes from the result file to add them to the export file.
  • Save results: by clicking this button the file is either sent to your ELN (please refresh the page) or saved locally.
