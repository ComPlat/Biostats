The History tab allows you to reproduce a previous analysis by loading its associated history JSON.

Step 1: Paste a valid history JSON string into the input box labeled History-JSON.
You can generate this string by exporting or saving a previous analysis session.
Step 2: Click Replay history to apply the full analysis to your current dataset.
Step 3: Confirm that you want to overwrite your current analysis in the confirmation dialog.

The application will then re-run each step of the analysis â€” including:

  • Creating intermediate variables
  • Applying filters
  • Adding new columns
  • Fitting statistical models
  • Running tests and creating plots

  • The current dataset must be loaded before replaying a history.
  • Replay will overwrite all existing results and settings.
  • Malformed or incompatible history JSON will show an error with the failing step number.

For reproducibility, a history JSON is always saved along with your results.
