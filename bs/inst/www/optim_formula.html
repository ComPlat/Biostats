1. Overview

The optimization module allows you to determine unknown parameters in a mathematical model so that the model best fits your observed data. For example, you can fit a curve of the form y = a * x + b to your data by estimating the optimal values of a and b.

The formula behaves like a standard mathematical function: the left side is the known outcome (e.g. y), and the right side defines the function you want to fit using predictors (e.g. x; Notably, x would be a column in your dataset) and parameters (e.g. a, b).

Parameters are not chosen from your dataset — they are automatically inferred through optimization.

2. Getting Started

  1. Select a column as your dependent variable (left side).
  2. Construct a formula on the right side using predictors and parameter names of your choice.
  3. Use operators like +, *, -, and / to define relationships.
  4. Set parameter boundaries and initial seed (random starting point) in the lower panel.
  5. Click Create Statistical Model to start the optimization.

3. Left Side of the Model

  • This is the known output (e.g., a column y from your dataset).
  • Use the dropdown to select it.

4. Right Side of the Model

  • Construct a function using dataset variables (predictors) and new parameter names (e.g. a, b).
  • You can write expressions such as a * x + b, c * log(x), or a / (1 + exp(-b * x)).
  • Edit the formula manually or use the provided variable and operator buttons.

5. Example: Fit a Straight Line

Goal: Fit a linear model y = a * x + b where y is measured data and x is a predictor.

Sample Data
x y

Model Formula

y ~ a * x + b

How it works
  • x comes from your dataset.
  • a and b are unknowns — the optimizer finds the best values for them.
  • You set value boundaries (e.g., a between 0 and 10).
Interpretation
  • The optimizer minimizes the difference between the predicted a * x + b and the actual y values in your data.
  • Once the model is created, it returns the best-fitting parameter estimates and optionally a LaTeX-rendered formula.
