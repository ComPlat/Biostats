2. Getting Started

To create your statistical model:

  1. Select a variable from the dropdown for the left side of your model.
  2. Choose predictor variables from the right side using the buttons provided.
  3. Add any necessary arithmetic operations using the operator buttons.
  4. Click the Create Statistical Model button to build your model.

3. Left Side of Statistical Model

This section allows you to define the left side of your statistical model, which typically represents the dependent variable (the outcome you want to predict).

Features:
  • Variable Selection: Use the dropdown menu to choose a variable for your model.
  • Create Model Button: Click this button to build the statistical model once you have selected your variables.

4. Right Side of Statistical Model

The right side represents the independent variables (predictors) that explain the outcome.

Features:
  • Variable buttons: Displays the selected predictor variables for your model.
  • Arithmetic Operators: Use the provided buttons to include mathematical operations between variables. Each operator button serves a specific purpose:
    • Add (+): Include an additional predictor variable.
    • Subtract (-): Remove a predictor variable from the model.
    • Multiply (*): Assess interactions between variables.
    • Colon (:): Include interactions between two variables.
    • Divide (/): Include nested effects that consider both variable levels.
  • Editable Code Area: A text area where you can manually edit the right side of the model if needed. This is useful for users who want to refine their formulas directly.

Interpretation
  • Intercept: The estimated weight when height is 0 is -100.14. This value is not meaningful physically but required mathematically.
  • height coefficient: Each cm increase in height is associated with ~1 kg increase in predicted weight.
  • p-values: Both terms are statistically significant.

Interpretation
  • Intercept: The expected price of a property when size and number of bedrooms are both 0. Not meaningful in practice, but mathematically necessary.

  • Bedrooms: Each additional bedroom increases the price by ~1.20 units, though this effect is not statistically significant (p > 0.05).

Model with Main Effects and Interaction Using *

Model: yield = α + β1(fertilizerLow) + β2(waterLow) + β3(fertilizerLow × waterLow) + ε

Sample Data
Yield Fertilizer Water

48LowHigh
53LowLow
Formula

yield ~ fertilizer * water

Model Output
Term Estimate => The ßs in the model Std. Error => Standard Error t-value => ßs / Standard Error p-value
Intercept => Yield when fertilizer = High and water = High 78.5 4.66 16.83 0.000073
fertilizerLow => Effect of Low vs High Fertilizer -37.0 6.60 -5.61 0.00496
waterLow => Effect of Low vs High Water -23.0 6.60 -3.49 0.0252
fertilizerLow:waterLow => Interaction: extra effect of both Low 28.0 9.33 3.00 0.0399
Interpretation
  • Intercept: Expected yield when both fertilizer and water are at their reference levels (High).
  • fertilizerLow: Applying low fertilizer instead of high reduces yield by ~37 units.
  • waterLow: Applying low water instead of high reduces yield by ~23 units.
  • Interaction: The combination of low fertilizer and low water increases yield by ~28 units *relative to what the sum of the main effects would predict*. This suggests a non-additive effect: the impact of fertilizer depends on the water level (or vice versa).
Alternative Formula Representation

yield ~ fertilizer + water + fertilizer:water

  • This version separates main effects and the interaction explicitly.
  • An interaction means the effect of one predictor depends on the level of the other.
