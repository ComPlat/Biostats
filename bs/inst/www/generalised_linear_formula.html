1. Overview

A formula describes how the columns of your dataset are mapped in the statistical model. For example the table shows three columns: CO2Uptake, Fertilization, and waterStress. Here the CO2Uptake would be the response variable, i.e., the left side of your model. The Fertilization and WaterStress would be the predictors, i.e., the right side of the model.

Generalised Linear Models (GLMs) extend linear models to handle non-normal outcomes, like:

  • Counts (Poisson, quasi-Poisson)
  • Binary data (yes/no, 0/1) using logistic regression
  • Proportions (binomial)

You must choose both a distribution family (e.g., binomial) and a link function (e.g., logit) when using a GLM.

2. Getting Started

  1. Select a dependent variable (left side).
  2. Select one or more predictors (right side).
  3. Use operators (e.g., +, *) to combine predictors.
  4. Choose a family and link function matching your data type.
  5. Click Create Statistical Model.

3. Left Side of Statistical Model

  • This is your response variable (e.g., 0/1 for logistic regression).
  • Use the dropdown to choose a column from your dataset.

4. Right Side of Statistical Model

  • Choose predictors by clicking on variable buttons.
  • Use the +, *, or : operators for combinations and interactions.
  • Edit the formula in the code area if needed.
