Split Dataset

This modal window enables users to select a subset of their dataset. The process involves two drop-down menus:

  • Column Selection: The first drop-down menu displays the column names from the dataset. Users can select one or more columns for splitting.
  • Group Selection: The second drop-down menu presents the unique values (levels) for the selected columns. These values are displayed below each respective column.

By selecting the desired groups, users can filter the dataset accordingly. Notably, if multiple columns are selected, the data is split sequentially based on the order of the selected columns.
